# Slim requirements for the phoneme -> text LoRA training run
# (Models/Llama.py + Data/phoneme_*.py only).
#
# Use this on the cluster instead of the full requirements.txt so we
# skip the heavy video / audio stack (mediapipe, decord, librosa,
# whisper, jax, ...). Mirrors the install block in newVALLR.ipynb cell 3.
#
# Install:
#     pip install -r requirements-llm.txt
#
# We deliberately do NOT pin torch / torchvision / torchaudio here:
# on Colab and on most HPC nodes a CUDA-matched build is provided
# (either pre-installed or via `module load`). Re-installing torch
# from PyPI usually breaks against the system CUDA. If you need to
# install torch yourself, do it FIRST against the right CUDA index,
# e.g.:
#     pip install --index-url https://download.pytorch.org/whl/cu121 \
#         torch==2.4.1 torchvision==0.19.1 torchaudio==2.4.1
# and only then `pip install -r requirements-llm.txt`.
